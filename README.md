# АСКАО B2B Site

Содержимое архива собрано по текущему canvas.

## Файлы

- `index.html` — текущая главная страница из canvas
- `services.html` — отдельная страница услуг с переключаемыми категориями

## Как открыть

1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. В верхнем меню нажмите `Услуги`, чтобы перейти на `services.html`.

## Важно

Страница услуг работает без Rider и без сервера. Категории переключаются через встроенный JavaScript.
